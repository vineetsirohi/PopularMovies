1. To compile this app, you will need the API key from themoviedb.org.

2. Make a java file 'TheMovieDbApiKey' in 'in.vasudev.popularmovies.model' package with a constant field 'VALUE' containing the value of your api key as shown below.


    package in.vasudev.popularmovies.model;

    public class TheMovieDbApiKey {

        public static final String VALUE = "YOUR_API_KEY";
    }
